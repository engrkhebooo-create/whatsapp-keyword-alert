# تنبيه الكلمات — مراقب إشعارات واتساب

تطبيق أندرويد يراقب إشعارات واتساب، ويبحث فيها عن كلمات مفتاحية تحددها أنت
(مثل: معلم، رياضيات، تدريس خصوصي...). عند العثور على تطابق، يقوم التطبيق فوراً بـ:

- 🔔 إصدار تنبيه (إشعار خاص).
- 🔊 صوت + 📳 اهتزاز.
- 💾 حفظ الرسالة داخل التطبيق.
- 📤 (اختياري) إرسالها إلى Telegram أو n8n / Webhook.

الهدف: عدم تفويت رسائل الوظائف أو إعلانات التدريس المهمة دون متابعة كل مجموعات واتساب يدوياً.

---

## كيف أحصل على نسخة APK للاختبار؟

التطبيق يُبنى تلقائياً عبر **GitHub Actions** (لا تحتاج تثبيت أي شيء على جهازك).

### الخطوات:
1. أنشئ مستودعاً جديداً على GitHub (مثلاً `whatsapp-keyword-alert`).
2. ارفع محتوى هذا المجلد إلى المستودع:
   ```bash
   git init
   git add .
   git commit -m "WhatsApp keyword alert app"
   git branch -M main
   git remote add origin https://github.com/<اسمك>/whatsapp-keyword-alert.git
   git push -u origin main
   ```
3. افتح تبويب **Actions** في المستودع → ستجد بناء "Build APK" يعمل تلقائياً.
4. بعد انتهائه (دقيقتان تقريباً)، افتح البناء → قسم **Artifacts** → حمّل `app-debug`.
5. فك الضغط، وانقل `app-debug.apk` إلى هاتفك وثبّته (فعّل "تثبيت من مصادر غير معروفة").

---

## التشغيل على الهاتف

1. افتح التطبيق "تنبيه الكلمات".
2. اضغط **"فتح إعدادات صلاحية الإشعارات"** وفعّل التطبيق في القائمة (Notification Access).
3. ارجع للتطبيق — يجب أن تتحوّل الحالة إلى ✅ مفعّلة.
4. عدّل الكلمات المفتاحية إن أردت (كل كلمة في سطر) واضغط **حفظ الإعدادات**.
5. (اختياري) أضف Bot Token + Chat ID لتيليجرام، أو رابط Webhook لـ n8n.

> ملاحظة: ليبقى التطبيق يعمل دائماً، استثنِه من موفّر/توفير البطارية في إعدادات الهاتف.

---

## البناء عبر Android Studio (بديل)

1. افتح المجلد كمشروع في Android Studio.
2. انتظر مزامنة Gradle، ثم Run ▶️ على هاتفك أو محاكي.

---

## بنية المشروع

| الملف | الوظيفة |
|------|---------|
| `NotificationListener.kt` | الخدمة التي تقرأ الإشعارات وتطابق الكلمات |
| `TextNormalizer.kt` | تطبيع النص العربي للمطابقة الدقيقة |
| `SettingsStore.kt` | حفظ الإعدادات والكلمات المفتاحية |
| `MatchStore.kt` | حفظ الرسائل المطابِقة محلياً |
| `Forwarder.kt` | الإرسال إلى Telegram / Webhook |
| `MainActivity.kt` | الواجهة والإعدادات |
